Announcement of phyclust
Instead of the irregular maintenance for the vignette, phyclust-guide.pdf, we may only update more examples and the newest documents on our website. Please check Phylogenetic Clustering at https://snoweye.github.io/phyclust/ . Some documents for main functions in the ../Documents/ subdirectory are provided for references and information for outsourced programs ms, seq-gen, baseml of PAML, and Hap-Clustering. Here are the descriptions for each file.
Readme File Description
00_Readme-phyclust-c.txt A text file for the library phylust-c, a kernel of this package, and basically it is in the format of DESCRIPTION file in R and records the version. Only the stable version of C library would be used to build the package.
00_Readme-ms.txt A text file for ms indicates the version and documentations.
00_Readme-seq-gen.txt A text file for seq-gen indicates the version and documentations.
00_Readme-paml_baseml.txt A text file for paml.baseml indicates the version and documentations.
License Description
ms-Readme.txt A text file, the original readme file in ms.
seq-gen-Readme.txt A text file, the original readme file in seq-gen.
paml_baseml-Readme.txt A text file, the original readme file in PAML.
tzeng-Readme.txt A text file, the original readme file in Hap-Clustering.
